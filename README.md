# vue-sortable-tree

### install
```
npm install vue-sortable-tree --save
```
global register
```
import SortableTree from 'vue-sortable-tree'
Vue.component(SortableTree.name, SortableTree)
```

partial register
```
import SortableTree from 'vue-sortable-tree'
# then in component
components: {
 [SortableTree.name]: SortableTree
}
```
